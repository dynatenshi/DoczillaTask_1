# Тестовые задания в компанию DOCZILLA

В данном репозитории содержатся *3* папки, в каждой из которой source-код заданий.

Для компиляции и запуска требуется запустить файл **_START.bat**, который есть в каждой папке.

Альтернативый вариант: 
1. В консоли установить директорию на основную папку задания, где содержится **pom.xml**
2. Ввести `mvn clean package`
3. После компиляции запустить **.jar** файл из папки target с помощью `java -jar ${путь.jar}`

## Задание 1 - сортировка жидкостей

В Main файле имеется несколько вариаций начальных состояний с разной сложностью.
Поиск решения осуществляется для каждого из этих вариантов, при помощи DFS с оптимизацией валидных вариантов.
