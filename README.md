# Тестовые задания в компанию DOCZILLA

В данном репозитории содержится source-код задания.

Для компиляции и запуска требуется запустить файл **_START.bat**.

Альтернативый вариант: 
1. В консоли установить директорию на основную папку задания, где содержится **pom.xml**
2. Ввести `mvn clean package`
3. После компиляции запустить **.jar** файл из папки target с помощью `java -jar ${путь.jar}`

---

## Задание 1 - сортировка жидкостей

В Main файле имеется несколько вариаций начальных состояний с разной сложностью.
Поиск решения осуществляется для каждого из этих вариантов, при помощи DFS с оптимизацией валидных вариантов.

После запуска программа последовательно выводит в консоль изначальное состояние бутылей, вычисляет выигрышную последовательность,
а затем выводит её, вместе с общей длинной последовательности и решенным состоянием головоломки. Запись каждого хода имеет формат (из -> в, кол-во).

Изначально поиск ограничен глубиной в 100, и если в рамках этого числа не было найдено решение, то такой вариант головоломки считается нерешаемым.
